# Clasificaci-n-de-localizaci-n-de-celular-de-prote-nas
Este apartado presenta el diseño, entrenamiento y evaluación de una red neuronal desarrollada en Python y PyTorch con el objetivo de clasificar la localización de proteínas celulares utilizando el conjunto de datos Yeast de la UCI Machine Learning Repository. El objetivo fue superar el 56.87 % de exactitud reportado por la UCI.
